About AI or Not

A machine learning platform that helps you distinguish real photographs from AI-generated images — built as a final-year university project.

Overview

Project Mission

As AI image generators become more sophisticated, verifying visual content authenticity is increasingly important. AI or Not provides an accessible, educational tool for detecting AI-generated images using classical machine learning.

Pipeline

How Detection Works

1. Image Upload

User uploads a PNG, JPEG, or WEBP image through the web interface.

2. Feature Extraction

OpenCV extracts ~205 handcrafted features: RGB/HSV color histograms, Canny edge density, Sobel texture gradients, and statistical color moments.

3. Logistic Regression

A trained logistic regression classifier predicts whether the feature vector represents a real or AI-generated image.

4. Result Display

The frontend shows the prediction, confidence score, analysis duration, and saves the result to local history.

Technology

Tech Stack

  • HTML5, CSS3, JavaScript (ES Modules)
